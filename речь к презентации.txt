Слайд 1. Титульный
Здравствуйте, уважаемые члены комиссии.
Я, Мищенко Виктор Сергеевич, студент группы 23П-2, представляю отчёт по производственной практике.
Тема моей работы — «Разработка и интеграция системы учёта компьютерной техники».
Руководитель практики — Калинин Арсений Олегович.
Практика проходила на базе Слободского колледжа педагогики и социальных отношений.

Слайд 2. Цель практики
Целью практики была разработка и интеграция системы учёта компьютерной техники для нашего колледжа.
Система должна была объединить в себе учёт кабинетов, компьютеров, пользователей и оборудования в единое целое.

Слайд 3. Архитектура системы
Для реализации был выбран язык C# и платформа WPF на .NET Framework.
В качестве системы управления базами данных использовался Microsoft SQL Server.
Также была выполнена интеграция с внешним API.
Все исходные коды сохранялись в системе контроля версий Git и загружены на GitHub.

Слайд 4. База данных
База данных содержит 13 таблиц.
Основные сущности — это кабинет, автоматизированное рабочее место, пользователь и заявка.
Все таблицы связаны между собой, что обеспечивает целостность данных.
На слайде представлена ER-диаграмма.

Слайд 5. Программа
Само приложение включает в себя несколько ключевых модулей.
Это личный кабинет пользователя, страница новостей, список кабинетов, просмотр компьютеров и детальная информация о каждом устройстве.
В системе реализовано разграничение прав для администратора, техника и преподавателя.
Также доступно изменение статуса компьютера и добавление новостей через интерфейс.

Слайд 6. Интеграция с API
В рамках практики была выполнена интеграция с внешним API — сервисом api.ipify.org.
Он позволяет получить публичный IP-адрес компьютера.
На слайде представлен код метода, который выполняет этот запрос, и подтверждение того, что полученный адрес действительно является публичным — он принадлежит провайдеру Cloudflare и находится в России, в Нижнем Новгороде.

В результате практики разработана и интегрирована система учёта компьютерной техники.
Все модули работают, тестирование пройдено, цель достигнута.
Система готова к эксплуатации в учебных целях.
Все материалы доступны в репозитории по ссылке на слайде.
